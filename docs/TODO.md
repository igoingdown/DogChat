# DogChat 后续待办清单

> 代码开发已完成，以下是上线前需要人工完成的步骤。

## 一、合规准备（前置依赖）

### 1.1 注册北京个体工商户

- [ ] 访问 [北京市企业服务 e 窗通](https://ect.scjgj.beijing.gov.cn)
- [ ] 注册账号，进入"个体工商户开办"
- [ ] 名称自主申报（建议：`北京市XX区犬趣信息技术工作室`）
- [ ] 经营范围勾选：`信息技术咨询服务`、`软件开发`、`技术服务`
- [ ] 经营场所：住宅地址 + 住所承诺申报制
- [ ] 上传身份证，电子签名
- [ ] 等待审核（1-3 个工作日）
- [ ] 下载电子营业执照

### 1.2 注册微信小程序

- [ ] 访问 [微信公众平台](https://mp.weixin.qq.com)
- [ ] 用新邮箱注册小程序账号
- [ ] 主体类型：企业 → 个体工商户
- [ ] 填写营业执照信息、经营者信息
- [ ] 支付 300 元微信认证费
- [ ] 等待认证审核（1-3 个工作日）

### 1.3 小程序 ICP 备案

- [ ] 在微信公众平台 → 设置 → 点击"去备案"
- [ ] 填写主体信息（自动带入）
- [ ] 填写小程序信息
- [ ] 负责人人脸核验
- [ ] 等待平台初审（1-2 天）
- [ ] 完成工信部短信核验（12381 发送验证码，24 小时内）
- [ ] 等待通管局终审（7-20 个工作日）

### 1.4 申请社交-笔记类目

- [ ] 微信公众平台 → 首页 → 小程序类目 → 添加类目
- [ ] 选择"社交" → "笔记"
- [ ] 上传 ICP 备案截图
- [ ] 等待审核（1-7 个工作日）

## 二、开发环境配置

### 2.1 微信开发者工具

- [ ] 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- [ ] 用小程序管理员微信扫码登录
- [ ] 导入项目，选择当前仓库根目录

### 2.2 配置项目

- [ ] 修改 `project.config.json` 中的 `appid` 为你的真实小程序 AppID
- [ ] 开通云开发：开发者工具 → 云开发 → 开通 → 创建环境
- [ ] 如果有多个云开发环境，可在 `miniprogram/app.js` 的 `wx.cloud.init` 中显式指定 `env`

### 2.3 部署云函数

- [ ] 在开发者工具中，分别右键以下聚合云函数目录 → "创建并部署：云端安装依赖"
  - `cloudfunctions/user`
  - `cloudfunctions/dog`
  - `cloudfunctions/friend`
  - `cloudfunctions/moment`
  - `cloudfunctions/walk`

### 2.4 创建数据库集合

- [ ] 进入云开发控制台 → 数据库
- [ ] 创建以下集合：
  - `users`
  - `dogs`
  - `friendships`
  - `moments`
  - `comments`
  - `likes`
  - `walks`

### 2.5 默认图片

- [x] `miniprogram/images/` 目录已包含以下占位图片，可后续替换为正式设计稿：
  - `default-dog.png` — 默认狗狗头像
  - `default-avatar.png` — 默认用户头像
  - `logo.png` — 登录页 logo
  - `tab-home.png`、`tab-home-active.png` — 首页 tab 图标
  - `tab-walk.png`、`tab-walk-active.png` — 约遛 tab 图标
  - `tab-profile.png`、`tab-profile-active.png` — 我的 tab 图标

## 五、已知问题 / 后续优化

| 问题 | 优先级 | 说明 |
|---|---|---|
| 图片内容安全审核 | 中 | `moment/create` 目前只有文本审核，图片审核需接入 `mediaCheckAsync` 异步轮询 |
| 二维码生成 | 中 | 当前用 canvas 绘制文字，应改用 `wxacode.getUnlimited` 生成小程序码 |
| 正式视觉资源 | 中 | 当前 `images/` 目录为可运行占位图，后续应替换为正式 logo、头像和 tabBar 图标 |
| 云函数冷启动 | 低 | 后期可用定时触发器保活 |
| 图片压缩 | 低 | 当前前端压缩，可优化压缩质量 |
